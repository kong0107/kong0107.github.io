## 豆知識

就算有血緣關係，只要沒有「同居」就不是民法所說的「家屬」。

{% blockquote 民法第1122條 %}
稱家者，謂以永久共同生活為目的而同居之親屬團體。
{% endblockquote %}

{% blockquote 民法第1123條第3項 %}
雖非親屬，而以永久共同生活為目的同居一家者，視為家屬。
{% endblockquote %}
