## 豆知識
18歲即可買公益彩券，但20歲才可買運動彩券。

{% blockquote "公益彩券發行條例 第9條" %}
發行機構、受委託機構及經銷商之負責人或員工，不得發售公益彩券或支付獎金予未滿十八歲者。
{% endblockquote %}

{% blockquote "運動彩券發行條例 第13條第1項" %}
發行機構、受委託機構及經銷商，不得銷售運動彩券或支付獎金予未成年人。
{% endblockquote %}

{% blockquote "民法 第12條" %}
滿二十歲為成年。
{% endblockquote %}
